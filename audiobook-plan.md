# Race Course Decision Making - Audiobook Production Plan

## ElevenLabs Limits:
- Free tier: ~10K characters/month
- Paid tier: ~30K-100K characters/month (depends on plan)
- Book total: ~300K characters

## Voice Selection:
- **Recommended:** Use a more professional/narration voice than Josh
- Options: Adam, Antoni, Arnold (mature, clear, audiobook-style)
- Josh is too upbeat for long-form narration

## Chapter Breakdown:

1. **Preface** - 1,809 chars (✓ Can do now)
2. **Prologue** - 5,725 chars (✓ Can do now)  
3. **Introduction** - 20,472 chars (Need 2-3 months on free tier)
4. **Chapter I: Philosophy** - 65,259 chars (6-7 months)
5. **Chapter II: Strategy** - 164,966 chars (16+ months!)
6. **Chapter III: Tactics** - 31,964 chars (3-4 months)
7. **Conclusion** - 10,162 chars (1 month)

## Recommended Approach:

### Option A: Incremental (Free Tier)
- Month 1: Preface + Prologue (~7K chars)
- Month 2-3: Introduction (split into 2 parts)
- Months 4-10: Chapter I (split into 6-7 parts)
- Months 11-26: Chapter II (split into 16 parts)
- Months 27-30: Chapter III (split into 3 parts)
- Month 31: Conclusion

**Timeline: ~31 months**

### Option B: Upgrade ElevenLabs Plan
- Creator plan: 100K chars/month = ~$22/month
- Could finish entire book in 3-4 months
- **Cost: ~$88 total**

### Option C: Hybrid
- Start with free tier (Preface + Prologue this month)
- See how it sounds
- Upgrade if quality is good

## Recommendation:
Start with Option C - create Preface + Prologue NOW using a professional voice to test quality.
